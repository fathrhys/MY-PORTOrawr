# Portfolio v2

Ini adalah iterasi kedua dari website personal saya. Awalnya menggunakan Prisma MySQL, namun di-refactor menjadi statis 100% menggunakan `.md` files.

## Fitur Utama
- Super cepat karena di-generate statis (SSG).
- Penulisan write-up sangat mudah melalui Markdown.
- Desain elegan dengan TailwindCSS Typography.

Project ini menjadi ajang uji coba integrasi data Markdown dengan `gray-matter` dan Next.js App Router.
